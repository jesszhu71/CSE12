Lab 4: Syntax Checker


DESCRIPTION:

In this lab, the user enters a text file into program argument, and the code will return any errors of mismatched braces. 


FILES:

test1.txt - This file contains a test file to test the code

INSTRUCTIONS:

This program is intended to run with MIPS Assembler in MARS. User enters a text file name into the program argument that they want checked for balanced braces/brackets/parentheses. MIPS will either print success or error or specifying the unclosed braces or where the mismatch error occurs.
